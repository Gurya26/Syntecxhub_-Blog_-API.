# 🚀 SyntecxHub Blog Platform

A Full-Stack Blog Website built using Node.js, Express.js, MongoDB Atlas, HTML, CSS, and JavaScript.

## 🔗 Live Website

https://syntecxhub-blog-api-2.onrender.com/

## 🔗 GitHub Repository

https://github.com/Gurya26/Syntecxhub_-Blog_-API

---

## ✨ Features

* Create Blogs
* Edit Blogs
* Delete Blogs
* Search Blogs
* Upload Images
* Dark / Light Theme
* Pagination (Previous / Next)
* Blog Sorting
* Author Name
* Date & Time Display
* MongoDB Atlas Integration
* Fully Responsive UI

---
